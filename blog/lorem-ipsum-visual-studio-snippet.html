To create a code snippet in Visual Studio (Visual Studio 2022 as of this post)

  1. Save the XML code below into a new file in a text editor of your choice as "lorem.snippet".
  2. Launch Visual Studio, then go to Tools -> Code Snippet Manager. Choose HTML as the language.
  3. Next, press the Import button to import the lorem.snippet file you created.

<?xml version="1.0" encoding="utf-8" ?>
<CodeSnippets xmlns="http://schemas.microsoft.com/VisualStudio/2005/CodeSnippet">
    <CodeSnippet Format="1.0.0">
        <Header>
            <Title>lorem ipsum 1</Title>
            <Author>wilcoforr</Author>
            <Description>First paragraph of lorem ipsum placeholder text, 542 characters long.</Description>
            <Shortcut>lorem1</Shortcut>
        </Header>
        <Snippet>
